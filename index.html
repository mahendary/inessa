MIHY-May I Help You

Nearby Businesses

Proximity based shopping and services platform where you can find all your local shops/stores/home services/ service providers. Browse and buy from their products and services range.

Omni Channel- Secured Chat App

Only platform that allows customers to chat with Vendors and Service Providers. All rich chat features with end-to-end encryption. Manage order workflow right within chat.

Single App for Customers and Businesses

Single application for Customers, Merchants, Home Services and Delivery partners. A business can also be a customer to another business. Prolific User Engagement.

One User - Multiple Sessions.

Same user can login from multiple mobile devices and web browsers at the same time. Continue chat/shopping from where you left from one device to another.

Product & Service Library. Bulk upload Products

Choose from built-in Mihy library with lakhs of products and services from 180+ types of business and services. Add products in bulk immediately at ease and faster.

Adboards, Not ads

Businesses can offer deals/discounts with Adboards. No annoying ads and promotion notifications to users. Customer shall not miss any deals from their favorite businesses.

No Catalogue Required

Create Raw Orders with Text, Photos, Audio and Video. Creates catalogue from Orders. Confirm prices and charges before payment.

Shopping list - Share, Quotation & Order

Create Shopping list. Compare products and services from nearby businesses. Get quotation from multiple merchants and place order.

Order Tracking

E2E Order tracking with Merchants, Services and CSD from order placement -> delivery -> return -> refund right within chat. Easy payment reminders and notifications.

Delivery Partners & Crowd Sourcing Delivery

Built in Delivery Management System for exemplary online delivery experiences for both Home delivery and Store pickups.

Easy and Secure Payments. Immediate Settlements & Refunds*

All types of payments are supported. UPI payments are supported* and are free. Payments to businnesses are automated for quick settlements.

Support for B2C and B2B - Wholesale and Retail business

Businesses can offer whole-sale prices to other specific businesses. Credit (Khata) account to trusted customers is available.

Chat Features

  • Conversations are encrypted and can be read by sender and recipient only. Mihy will not read any chat conversations.
  • Secured web sockets and scalable chat server. Chat messages formats stored in fast and efficient NoSQL dynamo db.
  • User typing status, online / offline/ last seen information.
  • Message Not sent / Sent / Read status
  • Attachments – Text / Emojis / Images / Links / Youtube urls / PDF / Audio / Video / Camera / Voice-recording and any file
  • Attachments – Mihy Contacts and Products
  • Pin/Unpin chats, Mute/Unmute Chats
  • Recent chat dashboard with last messages
  • Delete selected user chats
  • Forward one or selected message
  • Delete one/selected messages for me or for all, delete all messages
  • Auto delete messages (with attachments) after 30 days
  • Copy and paste message(s)
  • Reply to a message and swipe to reply
  • Catalogue order/raw items order/Order in Images/Order in Audio/Order in Video file
  • View order status, perform actions on order and auto update order in chat
  • Bill and quotation invoice in chat
  • Order amount or Khata (Credit) amount payment reminders
  • Preview – Image, Video and PDF file
  • Save attachments to gallery
  • Broadcasts : Create with multiple users with favourite name, Add/remove members anytime, Set a profile pic to broadcast, Send all types of messages (except orders) to broadcast
  • Groups : Create with multiple users with favourite name, Add/remove members anytime, Set a profile pic, Send all types of messages (except orders), Add/remove admin users, Exit from group, Group notifications
  • Mihy user in Phone address book is already a contact in Mihy app. No need to add explicitly.
  • Simultaneous chat by same user from multiple login sessions
  • Offline chat history
  • Auto connect to chat when connected to internet again (online)
  • Share text/image/files from other apps to Mihy contacts via chat
  • Chat toast notifications when online and offline
  • No explicit local storage of attachments
